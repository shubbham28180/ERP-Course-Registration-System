# ERP-Based Student Course Registration System

## 📌 Project Description
This project is a Python-based ERP system for student course registration.  
It allows students to view courses and register without duplication.

## 🚀 Features
- Student profile creation
- View available courses
- Register for courses
- Prevent duplicate registration
- Data stored using CSV file

## 🧩 Modules
- student.py → handles student data
- course.py → manages courses
- file_handler.py → handles CSV data
- main.py → main program

## 💻 Technologies Used
- Python
- Pandas
- CSV file handling

## ▶️ How to Run
1. Open terminal in project folder
2. Run:
   python main.py

## 📷 Output
Menu-driven system for course registration
